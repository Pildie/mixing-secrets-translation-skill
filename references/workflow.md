# Operational Workflow for Agent

1. **Source Intake:** Receive English segment.
2. **Technical Scan:** Identify critical terms (Phase, Polarity, Buss, Mud).
3. **UI Identification:** Block UI labels (Threshold, Attack) from translation.
4. **Drafting:** Translate into PT-BR using "mentor" tone.
5. **Humor Transposition:** Verify if British idioms were localized or visual absurdities preserved.
6. **Formatting Check:** Remove spaces in units (e.g., "3 dB" -> "3dB").
7. **QA Pass:** Run "Forbidden Terms" check.
8. **Final Output:** Present PT-BR segment.
