# Prohibited Terms List

- **inversão de fase** (when referring to Polarity Inversion)
- **comprimidor** (never use for compressor)
- **equalizamento** (incorrect form; use equalização)
- **monitoramento de áudio** (use "monitoração"; monitoramento sounds like security/IT)
- **baixo end** (incorrect calque; use região de graves)
- **trackeamento** (use gravação)
- **mixador** (use engenheiro de mixagem)
- **soco / murro** (use punch / impacto)
- **lama / lodo** (use empastamento)
