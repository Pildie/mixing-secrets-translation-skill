# Formatting Conventions

### 1. Units of Measurement
- **Rule:** No space between number and unit.
- **Standard:** 10kHz, 3dB, 5ms, 44.1kHz.
- **Forbidden:** 10 kHz, 3 decibéis, 10 KHz.

### 2. Software UI Labels
- **Format:** Keep in English, Bold.
- **Example:** "Ajuste o **Threshold** até que a redução de ganho atinja 3dB."

### 3. Software Menus
- **Format:** English, Italics, use `>` for hierarchy.
- **Example:** *File > Export > Audio Mixdown*.

### 4. Musical Notation
- Use Brazilian "Cifrada" nomenclature: **Dó, Ré, Mi, Fá** (not C, D, E, F) when describing notes in text.
- Example: "A nota **Fá** ressoa na sala."
