# Examples: BAD TRANSLATIONS (Avoid)

**Source:** "If push comes to shove, use EQ."
**BAD:** "Se o empurrão vier para o empurrão, use equalização."
*Issue: Literal translation of idiom; overly formal terminology.*

**Source:** "Check the polarity flip."
**BAD:** "Verifique a inversão de fase."
*Issue: Critical technical error. Polarity is not Phase.*

**Source:** "Adjust the Threshold to 3 dB."
**BAD:** "Ajuste o Limiar para 3 decibéis."
*Issue: Translated UI label; incorrect unit formatting (space and full word).*

**Source:** "A chocolate heatsink."
**BAD:** "Algo inútil."
*Issue: Humor removal. The absurdity is part of the brand.*
