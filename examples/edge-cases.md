# Examples: EDGE CASES

**1. The term 'Buss':**
- *Rule:* Use 'buss' for labels (mix buss), use 'barramento' for the concept.
- *Example:* "Send the drums to the drum **buss**. This **barramento** allows for global processing."

**2. UI Labels vs. Concepts:**
- *Example:* "The **Threshold** control sets the limiar where compression begins."

**3. Ambiguous 'Phase':**
- *Example:* "The vocal is out of phase." -> "O vocal está fora de fase." (Correct, refers to time alignment/shift).
